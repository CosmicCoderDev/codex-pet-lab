# Codex Pet Lab v0.4.0

## English

- Adds Rook, an original red-haired court ace, as a complete installable v2 pet.
- Includes Kairo and Rook release ZIP files with `pet.json` and `spritesheet.webp`.
- Adds 73-frame previews in the bilingual browser studio.
- Adds bundled-pet listing, one-command installation by pet ID, validation, and repeatable packaging.
- Adds public asset and contribution policies.

Install after cloning:

```powershell
.\scripts\install-pet.ps1 -PetId rook -Force
```
