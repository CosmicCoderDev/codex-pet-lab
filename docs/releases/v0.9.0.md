# v0.9.0 — Three-Pet Batch

This release adds three original, install-ready Codex v2 pets:

- **Shino** — a quiet shadow runner for stealthy refactors.
- **Aster** — a frost guardian who cools down hot code paths.
- **Nova** — a neon mech pilot ready for high-risk deployments.

Each package contains a validated `1536×2288` transparent WebP atlas with 73 frames: nine standard animation states and 16 clockwise look directions. All three passed structural validation, chroma-edge cleanup, three-reviewer blind direction QA, continuity review, and an independent final visual review.

The bilingual web studio and both README files now expose nine completed bundled pets. English remains the default language, with complete Simplified Chinese UI and documentation.
