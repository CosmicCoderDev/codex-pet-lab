# Codex Pet Lab v0.5.0

## English

- Adds Nebby, an original quiet cosmic cat, as a complete installable v2 pet.
- Includes Kairo, Rook, and Nebby release ZIP files with verified checksums.
- Nebby contains 73 animated frames: nine standard states and 16 clockwise look directions.
- Repairs per-frame left-running mirroring and validates the final loop through three-worker blind direction review and independent visual QA.
- Updates the English-first bilingual website, documentation, and bundled installer list.
