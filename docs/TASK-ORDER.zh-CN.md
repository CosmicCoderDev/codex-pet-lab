# 当前任务单

## 范围

在 `D:\dev\codex\codex-pet-lab` 内建立独立的 Codex 自定义宠物项目，并发布为 GitHub 公开仓库。

## 约束

1. 英文为默认语言，简体中文完整同步。
2. 只做原创宠物形象，不使用明星肖像或未经授权的角色。
3. 普通头像不能冒充可安装宠物；导出前必须通过真实图集校验。
4. 不修改 `D:\dev\codex` 下的其他项目。
5. 本轮只创建代码仓库，不自动部署 GitHub Pages。

## 验收

- `npm test`
- `npm run check`
- 浏览器检查中英文切换、四个预设、图集上传、状态预览和导出按钮状态
- 检查 Git diff 后提交并推送到公开仓库

## 工作模式

Local。该项目是全新独立目录，不需要额外 worktree。
