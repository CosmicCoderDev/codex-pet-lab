# Project Working Agreements

- This repository is a standalone static web tool. Do not modify sibling projects under `D:\dev\codex`.
- English is the default product and documentation language; keep Simplified Chinese translations in sync.
- Avoid runtime dependencies unless a feature cannot be implemented safely with browser APIs.
- Never label an image as installable unless it passes the Codex atlas contract.
- Validate with `npm test` and `npm run check` before publishing changes.
